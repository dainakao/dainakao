上方

演目
の

。

大入り

の

た
。

が
座っ
て
い
た
の

、
勘違い
で
前
の
客
が
握り飯
を
くれ
た
もの

、
食べ
て
しまう
。
別
の
席
で
は
尿意
を
感じ
て
いる
客
が
い
た
。
便所
に
立つ
と
席
を
取ら
れる
ので
、
動こ
う

も

。

、

て
いる

一升瓶

し
て
しまう
。
起き
た

は
目
や
鼻
に
染みる
と

…
。
もともと
は
「
子
ほり
角力
」

部
で
あっ
た
の
が
、
独立
し
て
演じ
られる
よう

なっ
た
。
相撲
に

する

し
た

で
ある
。
サゲ
は
なく
、
酒飲み
の

大騒ぎ
。
相撲
場
風景
で
ござい
ます
。
」

。
もともと

」

。
一升瓶

、

が
抜け
なく

五
代目
文枝
の
もの
で
ある
。
笑い
の
多い

、
よく
道頓堀

など

会場
で

て
い
た
。
松
鶴

が
酒
好き

、
酒飲み
の
描写
は
得意
で
あっ

、
小便

う

ぬけ
て

で
ある
。
下
ネタ

ある
が
、
松
鶴
は
カラリ
と
演じ
て
おり
、
ドタバタ

の
おかし
さ
が

て
いる
。
また

の
八
代目
雷門
助六
は
「
こり
相撲
」
の
題
で
演じ
た
。
スカ
トロ
を

と
し
て
いる
点
は
、
「
禁酒
関所
」
と

で
