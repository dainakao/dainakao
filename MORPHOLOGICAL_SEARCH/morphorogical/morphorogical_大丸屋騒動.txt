大丸
屋
騒動
（
だい

そう
どう
）

落語
の
演目
の
一つ
で
ある
。
安永

に
実際
に
京
で
起こっ

が

で
、
講釈
ネタ
を
落語
化
し
た

で
ある
。
上方
落語
に
は
珍しい
人情噺
（
厳密
に
は
サゲ
が
ある
ので
、
狭義
の
人情噺
で
は
なく
落とし
噺
だ
が
）
で
、
前半
部
の
宗
三郎
と
番頭
の
軽妙
な
やりとり
、
富永
町
で

へ

演出
、
後半
部
の
芝居
噺

の
演出
など
かなり

で
いる
。
サゲ
は
地口
落ち
。
妖
刀
村
正
、
愛想尽かし
、
狂乱
し
た
宗
三郎
が
祇園
二軒茶屋
の

の
輪
に

する

など
、

の
『
縮
屋
新
助
』
『
伊勢
音頭

（
油屋
騒動
）
』
の
パロディ
が
見
られる
。
はめ

と
の

に

を

、
歌舞伎
や
京
の
風景
・
風俗
に
精通
し
なけれ
ば
なら
ない
など
、

。

で

の

として

れ
て
い
た
。
戦後
は
長く
演者
が
い
なかっ
た
が
、
1970

に
入る
と
初代
森
乃
福
郎
や
2
代目
露
の
五郎
兵衛
が
相次い
で
高座
に
かけ
、
5
代目
桂
文枝
も
得意
と
し
た
。
文枝
は
1990
年
の
口演
で
芸術
祭
賞
を
受賞
し
て
いる
。
伏見
大手町
の
商家
大
丸屋

兵衛
の
弟
、
宗
三郎
は
、
祇園
の
舞妓
お
とき
を
愛人
に
し
た
こと
が
親戚
の
怒り
を
買い
、
お
とき
は
祇園
の
富永
町
に
、
宗
三郎
は
木屋
町
三
條
に

別居
する
。
兄
として
は
、
親類
を
説得
さ
せ
た
上
で
、
いずれ
は
晴れ
て
二
人
を
夫婦
に
する
算段
な
の
だ
が
、
宗
三郎
に
は
兄
の
思い
が
伝わら
ない
。
家
に
伝わる
妖
刀
村
正
を
床の間
に

て
、
番頭
の

、
無聊

。

の

、

、

三郎
は
、

の

を
ぬけだし

に
、

の

に

。
「
お

、
久し
ゅうこなんだ
。
わ
て
なあ
、
お前
に
逢い

て
なあ
。
」
「
ほんま
、
久しゅう
おす
なあ
。
ところで
あんた
は
ん
お
一
人
どす
か
。
」
「
せ

。
ちょっと
一

燗
け

かいな
。

に

と

て
な
。
」
「
お

。

、

で
宗
三郎
を

が

に
不信

を

さらに

三郎
に

が

、

を

。
お

。

、

。
