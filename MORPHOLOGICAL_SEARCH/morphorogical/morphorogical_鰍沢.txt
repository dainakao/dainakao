鰍沢
（

の

ひとつ

江戸
落語
で
広く
演じ
られる
。
成立
に

三遊亭圓朝

友人

「
酔狂
連
」

「
卵酒
・

毒消し
の
護符
」

で
即席
に
、
あるいは
一
晩
で
作っ
た
説
と
、
河竹
黙阿弥
による
後半
の
台本
が
現存
する
こと
から
、
黙阿弥
作
と
の
説
と
が
ある
。
主
な
演者
に
4
代目
橘
家
圓
喬
、
6
代目
三遊
亭
圓
生
、
林家
彦
六
、
10
代目

馬

ら
が

れる
。
身延山
（
＝
久遠寺
）
の

を

旅人
は
、
帰り
に
大雪
に
遭っ
て

で
道

迷い
、
偶然
見つけ

一軒家
に
宿
を
頼む
。
そこ

い
た
妙齢
の
美人
・
お

に
卵酒
を
すすめ
られる

飲み
、
話
を
する
うち
、
お
熊
が
かつて
は
吉原
遊廓
の
遊女
で

、
現在
は
猟師
の
妻
で
ある
こと
が
分かる
。
旅人
は
宿
の
礼
として
、
お

に
、
財布
の
大金
の
中
から
、
いくら
か
を
渡す
。
お

は
「
薪
を
取り

（
あるいは
、
酒
を
買い
に
行く
ため
）

て

する
。
旅人
は
疲れ
と
酔い
の
ため
に
、
横
に
なる
。
そこ
へ
お
熊
の
夫
が
帰っ
て
き
て
、
旅人
が
残し
た
卵酒
を
飲み
、
たちまち

。
帰っ
て
き

熊
は

「
旅人
に
しびれ
薬

酒
を

せ

殺し

金

。
その

、
すでに
毒
が
回っ
た
体
で
吹雪
の
舞う
外
へ
飛び出し
、
必死
に
逃げる
。
お
熊
は
鉄砲
を
持っ
て
旅人
を
追いかける
。
旅人
は
、
持ち合わせ
て
い
た
久遠寺
の
「
毒消し
の
護符
」

雪
とともに
飲み込み

その後
、
体

に
なる

、
川岸

追い詰め
られる
。

。
運

の
中
で
は
なく
、
岸
に
つない

あっ

今度

、
旅人
を
乗せ
た
まま
の
いかだ
が

。
お
熊
の

た

が
旅人

、
それ

近く
の
岩
に

。
急流
を
下る
うち
、
綱
が
切れ
て
いかだ
は
バラバラ
に
なる
。
旅人
は

一部

た
1

を

を
流れ

。
そのうち

は
、
お
熊
の

が

ない

流れ着き
、
窮地
を
脱する
。
「
この

た

、
お

。
お

（
＝
お

）
