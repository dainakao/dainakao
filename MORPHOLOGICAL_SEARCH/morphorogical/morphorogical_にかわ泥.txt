仏師
屋

東京

。

盗人
』

東京
で
は
『
置
泥
』

『
夏
泥
』
）
と
似る
。
6
代目
笑
福
亭
松
鶴
が
得意
と
し
た
。
現在
で
は
6
代目
笑
福
亭
松
喬
ら
松
鶴
門下
が
多く
演じ
て
いる
。
演者
は
まず
「
十
両
の
額
の

を
盗ん
だら
斬首
刑
に
処さ
れ
た
」
という
、
江戸
時代
における
窃盗
犯
の
刑罰
に
少し
触れる
（
「
太い
奴
␣
どうして
九
両
三
分
二
朱
」

川柳
を

する
など
）
。

、
盗人
が

（

を

）

。
盗人

仏師
屋

、

を

「
金
を
出せ
」

、

屋
は

こと
も

も
なく

「
そこ
の

一
両
二

が
ある

、
持っ
て
行き
」

。
調子
の
狂っ

外

そう

た

。

「
何

。
「
おまえ

大和

『
首
、
継い
で
くれ
』

ゅうて

て

た
びん
ずる
さん
の
首
、
継い
どい
た
もん

や
がっ
て
!」
盗人

屋
の
あまり

剣幕
の
前
に
平謝り
する

わ

接着
剤
）

、
仏像
の
首

に

仏像
の
首
は
再び
つながる
。
「
しかし
、
こん

二

何
ぬかし

ン
ねん
。
さっさと

って

ね
」

は

を

、
に
かわ
の
入っ

鍋
を

う

「
なん

鍋
ェ

。
十
