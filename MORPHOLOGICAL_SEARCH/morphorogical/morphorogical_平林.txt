（
ひ

、

。

（
じ

）

（

）

演題

で

られる
。

は
『

睡
笑
』

一

「
推
は
違う
た
」
。
また
、
名
の
読み方
を
巡る

を
描い
た
類似
の
民話
が
各地
に
残る
。
10
分
程度
の
短
さ
の
中
に
、
多く
の
クスグリ
が
含ま
れる
構成
で
あり
、
若手
が
鍛錬
の
ため

演じる
「
前座
噺
」
の
ひとつ
として
知ら
れる
。
芸歴
を
経
て
も
好ん
で
演じる
演者
は
東西
とも
多く
、
2
代目
立花
家

橘
に
、
ゆっくり
し
た
口調
で
15

以上
かけ
て
演じ
た
音源
が
残る
。
商家
の
丁稚
の
定吉
は
、
（
東京
で
は
平河
町
、
上方
で
は
本町
に
住む
）
医師
の
「
平林
（
ひ
ら
ばやし
）
」

に

に
たずね
、
手紙
を
届け
、

を
もらっ

れる
。
定吉
は
、

ない

「
ヒラバヤシ
、
ヒラバヤシ
」

繰り返し
ながら
（
あるいは
、
お
使い
の
不平
を
ぼやき

）
歩く
が
、
結局
忘れ
て
しまう
。
定吉
は
思い出す

、
手紙
に
書か
れ

「
平林
」

、
そもそも

が
でき
なかっ

気づく
。
そこで
、

、
「
平林
」

。
最初

「
それ

安心

は
知り
ませ
ん
か
?」

、

、

『

』

『
林
』

字
は
リン
と

。

「
ヒラリン

の
お
宅
は
知り
ませ
ん
か
?」

、

）

。

ヤシ

定吉
