品川
心中

が

品川

。
前半

の

。
現代
で
は
前半
のみ

話
で
終了

せ
、
後半
の
下げ
の
部分
まで
の
話
を
する
やり手
が

い
ない
。
『
井関
隆子
日記
』
天保
11

（

）
の
2

の
条
に
、

と

れる

が
ある
。
また
、

「
松島
心中

。
ここ

は

を

、
後半
を
下

。
品川
の
女郎
「

」

、

が

ない

下
の

、

。
1

な

か

。
なじみ
の

から

を

、

こと

。
早速

を

お

。
翌日

晩
、

を

を

が
嫌がる

、
外
の

から

を
する
こと
に
する
。
桟橋
で
なかなか
飛び込も
う
と
し
ない
金蔵
を
お
染
が
突き落とし
、
自分
も
飛び込も
う
と
し
た
ところ
に
、
店
の
若い衆
が
「
金
が
出来
た
」

を
伝え
に
来る
。
お
染
は
死ぬ
の
が
馬鹿馬鹿しく
なっ
て
店
へ
戻っ
て
しまう
。
遠浅
だっ
た
ため
死に
そびれ

は

の
ところ
へ
行く
が
、
親方
の
家
で
は
博打
を
し
て
おり
、
戸
を
叩く

「
役人
だ
」

し
て

。
尋ね
て
き
た
の
が
金蔵
と
分かり

する
が
、
1
人
びく
と
も
し
ない
者
が
い
た
。
その
者
を
褒める
と
「
いや
とっくに
腰
が
抜け
て
おり
ます
」
。
翌朝
、
金蔵
が
親方
に

を
話し
、
怒っ
た

は

とともに
、
お
染

の

を

。
金蔵
は
、
お

を

て

、

「
白い

が
食い
てえ
」

、

。
しばらく

、
お
染
を
訪ね

来
た

が
ある

店
の
者
が
呼び
に
来る
。
出
て

、

と

という
二
人
連れ
が

おり

金蔵
の
通夜
に

て
もらい
たい
、

いう
。
驚い
た
お

が
、
そんな
はず
は
ない
、
と
、
親方
を
連れ
て
部屋
に
戻る

金蔵
の
姿
は
なく
、
蒲団
に

の

て

。
親方
は

、

は

、

、

。

。
