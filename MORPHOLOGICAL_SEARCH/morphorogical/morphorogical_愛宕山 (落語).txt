『
愛宕山
』
（
あたご
や
ま
）
は
、
古典
落語
の
演目
。
上方
落語
の
演目
の
一つ
で
ある
。
春山
の
ピクニック
を
描い
た
華やか
な
噺
で
、
京都
の
旦那
と
大阪
出身
の
幇間
（
太鼓
持ち
）
と
の
ユーモラス
な
やり取り
が
見どころ
で
ある
。
山
行き
の
足取り
や
番傘
で
の
飛び降り
、
谷
底
から
飛び上がっ
て
くる
シーン
など
要所
で
派手
な
アクション
が
入る
ため
、
長時間
の
話芸
とともに
相当
の
体力
が
要求
さ
れる
大
ネタ
で
ある
。
上方
で
は
3
代目
桂米朝
や
2
代目
桂枝雀
（
英語
落語
でも
）
、
桂
吉
朝
が
よく
演じ
た
。
また
江戸
落語
に
も
輸出
さ
れ
、
8
代目
桂
文楽
の
口演
は
非常
に
高い
評価
を
受け
た
。
上方
で
現在
演じ
られ
て
いる
の
は
、
3
代目
桂米朝
が
文
の
家
かしく
（
後
の
3
代目
笑
福
亭
福松
）
に
稽古
を
付け
て
もらっ
た
もの
と
、
5
代目
桂
文枝
が
師匠
4
代目
桂
文枝
から
稽古
を
付け
て
もらっ
た
ものの
二
通り
が
あり
、
演出
方法
も
少し
異なっ
て
いる
。
(
以下
は
上方

で
の

に
準拠
する

一
八
と
茂

大阪

を

京都

。

、

「
気候
も
いい
ので
野
駆け
（
ピクニック
）
を
しよ
や
ない
か
」

発案
し
た
ので
、
芸妓
や

、
お茶屋
の
女将
ら

愛宕山
へ
登る
こと
に
なっ
た
。
大阪
に
は
山
が
ない
から
山登り
は
大変
だろ
う
と
からかう
旦那
。
一
八
は
「
大阪
に
も
山
は
ある
」

真田山
・
茶臼山

天保山

を

「
そんな
もの
は
地べた

デンボ
（
でき

）
」

。
頭

高い

お
ま

、
二つ
重ね
て
ケンケン
で
上がっ

」

。
旦那

「
それなら
みんな
の
荷物
も
持っ
て
登っ
て
来い
」

の

、
その他

登っ

。
「
大阪
を
しくじる
ん
や
なかっ
た
」
と
悔やむ

二
人
だ
が
、
みんな
の
荷物
を
持っ
て
後
を
追いかけ
た
。
が
、
慣れ

に

に

し
て

。
一同
が

た

に
しよ
う

旦那
が

し
た
。

は

、

一

、

「

」

。

。
「

「

」
「

かわらけ
を
的

投げ入れる
旦那
。
一
八
は
見よ
う

真似
で
チャレンジ
する
が
全く
命中
し
ない
。
挙句

「
大阪
の

かわらけ

なし
ょうむないも
ン
投げ

。

こと
が
あっ
たら
と
思
て
持っ
て
き
た
」

、

昔
の

を
20

で

。
谷
底
の

を

「
これ
が
本当
の
散財
、
胸
が
スッ
と
し
た
」

言っ
た

に
、
「
あの
小判
は
ど
ない
なり
ま
ん
ねん
。
」

一
八
が
尋ねる
。
「
放っ
た
ん

から
拾っ
た
人
の
もの
」

の

、
一生懸命
手
を
伸ばし
た
が

。
茶店

に

、

を

替え

ある

ある

。

、

。

一瞬

そう

。
そこで

さ
れ

。
20

その

。

。

、

。
そして
力一杯

を

、
十分

せ
て

、
シュ
～
ッ

の
上

。
「
旦
さん
、

。

。
噺家

。
8

の
「
愛宕山
」

、
