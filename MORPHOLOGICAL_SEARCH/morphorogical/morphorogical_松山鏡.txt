松山
鏡
は
、

鏡
（
まつ
や
ま
かがみ
）
は
古典
落語
の
演目
の
一つ
。
原

は
、
古代
インド
の
民間
説話
を
集め
た
仏典
「
百
喩
経
」
、
第
三
十
五
巻
の
「
宝
篋
（
ほう
きょう
）
の
鏡
の
喩
（
たとえ
）
」
。
主
な
演者
として
、
8
代目
桂
文楽
など
。
「
無
二
膏
や
万能
膏
の
効き目
より
、
親孝行
は
なん
に
つけ
て
も
…
」
舞台
は
越後
の
松山
村
。
両親
が
死ん
で
十
八
年間
、
ずっと
墓参り
を
欠かし
た
事
が
ない
正
助
という
男
が
、
お上
の
目
に
留まり
ご
褒美
を
頂戴
する
こと
に
なっ
た
。
村役人
に
付き添わ
れ
、
役所
に
出頭
し
て
き
た
正
助
に
、
地頭
が
何
か
欲しい
物
は
ない
か
と
質問
。
␣
この
正
助
は
無欲
な
男
で
、
「
自分
は
当たり前
の
こと
を
し
た
まで
」
と
地頭
が
あげ
た
ご
褒美
を
すべて
辞退
し
た
。
それでも
何
か
し
て
あげ
たい
地頭
は
、
「
どんな
無理
難題
で
も
ご
領主
さま
の
ご
威光
で
かなえ
て
とら
す
ので
、
何
なり
と
申せ
」
と
質問
。
␣
それ
に対する
、
正
助
の
答え
は
意外
な
もの
だっ
た
。
「
おとっ
つ
ぁまが
死ん
で
十
八
年
に
なる
が
、
夢
でも
いい
から
一
度
顔
を
見
たい

思っ
て
いる
ので
、
どうか
おとっ
つ
ぁまに
一目
会わ
せ
て
ほしい
」
地頭
は
唖然
。
しかし
、
正
助
の
純粋
な
気持ち
に
感銘
し
、
何とか
叶え
て
あげ
たい
と
思案
し
た
。
名主
の
権右衛門
に
訊ねる

、
正
助
の
父親
は
四
十
五
で
他界
し
、
しかも
顔
は
せがれ
に
瓜
二つ
だ
と
いう
。
␣
これ
で
解決

を

は
、
家来
に
命じ

鏡
を
一つ
持っ
て

させ
た
。
地頭
に
言わ
れる

、
正
助
が
鏡
の
中
を
覗く
と
…
？
「
おとっつぁん
！
？
」
この
松山
村
は
田舎
と
言う
事
で
、
まだ
『
鏡
』
という
もの
を
誰
も
見
た

ない
。
正

も
映っ
て
い
た
自分
の

を
見
て
、
おやじ
が
映っ
て
いる
と
勘違い
。
感激
し
て
泣き
だし
た
。
その

を
見
て
い
た
地頭
は
、
自ら

を
取っ
て
鏡

箱

【
子
は
親
に
␣
似
たる
もの
を
ぞ
␣
亡き人

␣
恋しき
とき

鏡

ぞ

】

を

、
「
余人
に
見せる
な
」
と
言っ
て
下げ渡す
。
それから
と
言う
もの
、
正

は
納屋
の
古
葛籠
の
中
に
鏡
を

、
女房
に
も

に
し
て
、
朝夕

込ん
で
は
挨拶
を
し
て
い
た
。
␣
そんな
亭主
の

を
、

の
お

が

に

亭主
の

。
「

、
この

！
？

ない

、

た

を
夫

と

。
嫉妬

。
その

。
両方

。
「
ふふふ
、
正

、
お

よ
、
けんか
せ
ねえ
が
ええ
よ

。
お
め
え
ら

えれ

し
た

、

の

って

に

「

い

が
あっ

の

ない
。
この

、

た
「
ありがたい

」

、

て

た

『

』

』
、
『

四

』
など

ある
。
