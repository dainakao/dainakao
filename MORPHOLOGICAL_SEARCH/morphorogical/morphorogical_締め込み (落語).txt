締め

、
古典

ひとつ

同

、
東京
落語
で

。
この

、

盗人

仲裁
（
ぬ
す
と
のち

、

（
享
和
2

）

出版
さ
れ

『
新撰

』

末
しら
浪
」

夫婦

を

する

。
盗人
の
挨拶
（
ぬ
す
と
の
あいさつ
）
と
も
。
主
な
演者
に
、
初代
桂
春
団
治
、
5
代目
桂
文枝
ら
が
知ら
れる
。
時
の
氏神
（
とき
の
うじ
が
み
）
と
も
。
演目
の
成立

は
、
侍
が
ヤカン
を
盗む
という
ストーリー
だっ
た
が
、
上方
の
『
盗人
の
仲裁
』
の

が
移入
さ
れ
、
現在
の
演じ
方
が
成立
し
た
。
主
な
演者
に
、
4
代目
三遊
亭
圓
生
、
3
代目
柳家
小さん
、
5
代目
古今
亭
志
ん
生
ら
が
知ら
れる
。
長屋
の
、

さ
れ
て
い
ない

の
留守
を

、

が

、

そこ

の

『
締め
込み
』

は
八

）

帰っ
て
くる

が
聞こえ
て
き
た
ので
、
泥棒
は
裏口
から
逃げよ
う
と
する

、
戸
を
開ける

高い
塀
が
立ちふさがっ
て
い

、
裏口

なかっ

、
とっさ

台所

、

の

「
あの

、

知ら
ぬ

間男
を
作っ
て

荷物
を
まとめ

を
しよ
う

し
て
いる

、

。

が

て
くる

出
て

たけれ
ば
出
て
行け
」

。
事情

は

の

れ

い

。

は

私
の

。

、

!!……

もう

。
すると

ず
、
「
表
