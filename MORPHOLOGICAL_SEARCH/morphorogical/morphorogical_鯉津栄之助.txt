鯉
津

（
こいつ
ええ
の
すけ
）
は
、
上方
落語
の
演目
の
一つ
。
原

は
、

4
年
（

）

「

童子
」

はやり
」

の
旅
』

『
伊勢
参宮
神乃
賑
』
）
の
一
編
。
伊勢参り
の
途中
、
伊賀
名張
へ
と
やって来
た

と
清
八
。
おりしも
領主
の
鯉

氏
に
『
栄之助
』

ご

が
ご

なさっ

言う

で
、

の
名前
で
ある
【
鯉
津
栄之助
】
の
名前
に

『
こいつ
ぁええ
』

を
使わ
ぬ
よう
、
忠告
する
ため
の
関
が
作ら
れ
て
い
た
。
「
通行

を
所持
し
て
おる
か
？
」

に
なっ
て
いる
と
は
知ら
ない
二
人
、
あいにく
手形
を
用意
し
て
い
し
なかっ
た
。
「
本来
なら
通す
こと
相
なら
ぬ
が
…
おめでたい

故
、
芸
を
見せれ
ば

て
やる
」

？

て

、

。

。

、

。
しばらく

、

。

、

の

き

。

、

三

を

？
」
␣
「
一杯

の

を
聞く
…
こいつ
ぁええ
！
」
途端

が
数

で
き
て
、

グルグル

に
縛ら
れ
て
しまっ
た
。
「
アタタタタ
…
何
し
ます
ん
が
な
！
？
」
␣
「
黙れ
！
␣
み
ども
が
店
の
前
を
通り
し
おり
、
その
方
、
いま
『
こいつ
ぁええ
』

申し
た
な
！
？
」
喜六
は

ない
。
代わり

が

。

、

。

三

を

？

その

三
