宮戸

（

わ
）

の

。
一般

2
場面
の
うち

前半

、
お

半

ち

半

（

はな
はん
し
ち
）

と
なる
こと
も
ある
。
宮戸
川
と
は
、
現在
の
隅田川
の
うち
、
浅草
周辺

の

で
ある
。
前述
の
とおり

、

川
は

に

し
ない
。

の
「
お花
」

「

」

は
、

で

た
と
さ
れる

を

し
た

や

（
『
長町
女

』

）

れ
た
もの
。
後半
部分
を

て

と
長時間

、
（
夢
オチ

は
いえ

少ない
。
3

小網

を

て
い

、
家
に

て
もらえ
ず
、
締め出し
を
食っ
て
しまう
。
すると
、
友人

で
カルタ
を
し
て
い

帰り

遅く
なり
、
同じ

を

しまっ

・
お

と
行き
会う
。
半
七
は

を

の

。
お

私
も

」

、

は

「

れる

がち
な

。
うぶ

七

「
叔父
に
いい

だ

れる

、
どう

ない

。

つい
て
き
て

。

、

七
と
お花
は

に

を

2

れる
。
ふたり

に

、
お

は

七

で
も
ない

を

。
そのうち
、
雷
が

は
半
七
の

。

はだけ

お

を

…
…

る

は

。

半

「

。
