ろくろ首
（
ろくろく
び
）
は
、
古典
落語
の
演目
の
一つ
。
ろ
くろ
っ
首
（
ろ
くろ
っ
く
び
）
と
も
発音
・
表記
さ
れる
。
同
発音
で
ろくろく
首
の
表記
も
み
られる
。
妖怪
の
ろくろ首
と
、
「
首
」
を
用い
た
慣用
句
を
題材
に
し
た
滑稽
噺
。
元
は
幕末
の
万延
頃
に
成立
し
た
上方
落語
と
み
られ
て
い
て
、
東京
で
演じ
られ
た
最初
期
の
例
として
は
、
1905
年
（
明治
38
年
）
に
初代
三遊
亭
圓
左
が
演じ
た
記録
が
残る
。
他
の
主
な
演者
に
、
上方
で
は
5
代目
桂
文枝
、
2
代目
桂
ざこ
ばら
が
知ら
れる
。
東京
で
は
8
代目
林家
正蔵
、
3
代目
桂
三木助
、
10
代目
柳家
小
三
治
ら
が
いる
ほか
、
代々
の
柳家
小さん
が
家伝
の
ネタ
に
し
た
こと
で
知ら
れる
。
独身
の
男
（
喜六
あるいは
与太郎
の
キャラクター
。
東京
で
は
「
松
公
」
など
の
名
が
与え
られる
）
が
、
隠居
（
上方
で
は
甚兵衛
、
東京
で
は
「
岩田
の
隠居
」
）
の
自宅
を
訪ね
た
ところ
、
縁談
を
持ちかけ
られる
（
あるいは
、
男
の
方
から
見合い
話
を
頼む
）
。
隠居
は
「
婿養子
に
行く
気
は
ない
か
。
さる
資産
家
の
娘
で
、
器量
（
＝
容貌
）
も
よい
。
ただし
、
言っ
て
おか
なけれ
ば
なら
ない
欠点
が
ある
」
と
話す
。
男
は
いぶかし
がり
、
「
実は
子供
を
宿し
て
い
て
、
面倒
を
解決
する
ため
に
私
を
あてがお
う
という
の
でしょ
う
。
もしくは
、
会う
人
ごと
に
ある
こと
ない
こと
を
言いふらし
て
歩き
、
もめごと
を
作っ
て
喜ぶ
の
でしょ
う
」
など
と
問う
。
隠居
は
「
そう
で
は
ない
。
毎晩
午前
2
時
ごろ
に
なる
と
、
首
が
、
シューッ
、
と
伸びる
の
だ
。
これ
まで
何
度
も
結婚
し
た
ものの
、
その
たび
に
婿
に
逃げ
られ
て
しまい
、
困っ
て
いる
という
の
だ
」
と
告げる
。
男
は
「
それでは
ろくろ首
で
は
ない
か
」
と
気味悪
がり
、
一
度
は
渋る
が
、
「
夜中
は
寝
さえ
すれ
ば
、
伸びる
首
を
見
ず
に
すむ
だろ
う
（
あるいは
、
知人
に
縁談
が
行き
、
女
を
取ら
れる
の
は
面白く
ない
）
」
と
思い直し
、
婿入り
を
決意
する
。
男
が
、
娘
の
家
で
行う
挨拶
の
形式
を
覚え
られ
ない
ため
、
隠居
は
案じ
、
男
の
足
の
指
（
あるいは
、
ふん
どし
）
に
ひも
を
結び
、

が

引っぱれ

（
※
指
の
場合

別々
の
ひも
、
ふん
どし
の
場合
は
回数
）
、
男
が
「
左様
左様
」
、
「
ご
もっとも
ご
もっとも
」
、
「
なかなか
」

返事
する
よう
取り決める
。
隠居
は
男
を
羽織
袴
に
着替え

、
娘
の

て

。

の
乳母
が
応対

、
隠居

。
男
は
慣れ
ない
正座
の
ため

足
が
しびれ
、
体
を
しきりに
動かし
始める
。
それ
に
合わせ

結ん
で
い
た
ひも
が

、
娘

それ

。
男

が

いる

し
、
それ

合わせ
て
しゃべり
だす
。
「
なかなか
。
さよう

ご
もっとも
。
ご
もっとも
なかなか
…
…
」

（
※

、
日

ある
）

その

、
男
は
そば
で
眠る
娘
の

が
気
に
なり
、
寝付く
こと
が
でき
ない
。
男
は
、
娘
の
首
が
伸びる
の
を

し
て
しまい
、
恐怖

を
飛び出す
。
男
は
隠居
宅
の

を
叩き
、
「
首
が
伸び
た
」

ながら
転がり込む
。
「
伸びる
の
を
承知
で
行っ
た
の
だろ
う
」
「
まさか

（
し
ょにち
）

思わ
なかっ
た
」
「
芝居

ない
の
だ
から

も

も
ない

う
。
もう一度

戻り
なさい
。
お嬢さん
が
おまえ
の
帰り
を
、
今
か
今
か

待っ
て
いる
」
「
怒っ
て
い
ない
でしょ
う
か
。
どんな
風
に
待っ
て
いる
でしょ
う
か
」
「
首
を
長く
し
て
待っ
て
いる
」
