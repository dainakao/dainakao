。
三ノ輪

。

が
切符
の
検札

し
て
いる
。

、

の

と

を

ます
。
」

、

。

は

だ
。
」

じゃあ
、
買っ
て
いただき
ます
。
」

タダ
だ
ん
べ
え
。
」
「
タダ
じゃ
ござん
せ
ん
。
」
「
は
あれ
。
東京

は
生き馬

を

とこ
だ
と

て
い

・
・
・

、
抜い

べ
え
。

ませ
ん
。

。

。

。

。

。
」

ない

。
」

で
え
。

！
」

。
」

そう

。

。

。

。

。

酒
だ

。

。

。

。

よ

。
」

件

「
・
・
・
う
ー
い
。

！
蟹
！

ア
あり
ませ
ん
。
ここ

何
