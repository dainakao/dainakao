壺
算
（

）

古典
落語
の

の

。
原

は
、
延
享
4
年
（
1747

）

出版
さ
れ
た
笑話

「
軽口
瓢
金
苗
」

一

ある
「
算用
合
て
銭
たら
ず
」
。
元々
は
『
壺
算用
』
という
上方
落語
の
演目
。
主
な
演者
として
、
東京
の
6
代目
三
升
家
小勝
や
柳家
権

楼
、
上方
の
2
代目

仁鶴

いる
。
␣
「
二

」

の

公
。
しかし
この

は
「
黙っ
て
い
た
方
が
利口
に
見える
」

まで
言わ
れる
ドジ
。
おかみ
さん
に
言わ

、
買い物
上手
と

の

を
求め
に
訪れ
た
。
␣
そんな

の

を
快く

た

分
。
だが
、
瀬戸物
屋
を
訪れ
た

が

を
つけ
た

は
、
なぜ
か
半分
の
一
荷
入り
の

だっ
た
。
この

の

は
本来
三
円
と
五
十
銭
だっ
た
の
だ
が
、
兄貴
分
は
瀬戸物
屋
を
太鼓
持ち
も
顔負け
の
口調
で
おだて
上げ
て
五
十
銭
値引き
さ
せ
て
しまっ
た
。
何
度
も
文句
を
言い
かける
吉
公
を
制し
、
兄貴
分
は
三
円
を
置く
と
一
荷
入り
の
水がめ
を
吉
公
に
担が
せ
て
店
を
出
て
しまう
。
「
俺
の
買い
たい
の
は
二
荷
入り
の
壺
だ
よ
」
「
任せ
て
おけ
。
それ
じゃあ
、
また
瀬戸物
屋
へ
引き返す
ぞ
」
そして
兄貴
分
は
瀬戸物
屋
に
こう
告げる
。
「
実は
手違い
が
あっ
た
ん
だ
よ
。
こいつ
の
買い
たかっ
た
の
は
二
荷
の
かめ
な
ん
だ
が
、
コイツ
が
度忘れ
し
て
一
荷
入り
の
水がめ
を
買っ
ちゃっ
た
ん
だ
」
␣
瀬戸物
屋
に
二
荷
入り
の
値段
を
訊く
と
、
一
荷
入り
が
三
円
五
十
銭
な
ので
、
二
荷
入り
は
その
倍
の
七
円
に
なる
と
いう
。
兄貴
分
は
瀬戸物
屋
に対し
、
さっき
五
十
銭
値下げ
し
た
の
を
引き合い
に
出し
て
こっち
も
値下げ
し
て
くれ
と
おだて
上げ
ながら
交渉
し
、
とうとう
七
円
の
水瓶
を
六
円
に
まけ
させ
て
しまう
。
␣
それでは
お代
の
六
円
分
を
という
瀬戸物
屋
に
、
兄貴
分
は
「
水がめ
は
二つ
も
いら
ない
から
、
さっき
買っ
た
の
を
下取り
し
て
くれ
ない
か
」
。
それ
を
買っ
た
値段
の
三
円
で
引き取る
という
瀬戸物
屋
。
兄貴
分
は
「
さっき
の
一
荷
入り
を
下
取っ
て
三
円
、
それから
最初
に
渡し
た
三
円
。
足し
て
六
円
だ
」
と
言い
、
吉
公
に
二
荷
入り
の
水がめ
を
担が
せ
て
店
を
出
て
いく
の
だっ
た
。
␣
ドジ
の
吉
公
でも
この
トリック
に
は
さすが
に
気が付い
た
。
「
ハハハ
、
あの
ご
主人
馬鹿
だ
ねぇ
」
「
声
出す
な
、
気付か
れる
ぞ
」
何
か
おかしい
と
思い
、
慌て
て
呼び止める
瀬戸物
屋
。
それ
に対して
兄貴
分
は
さっき
と
同じ
話
を
繰り返し
た
。
また
慌て
て
呼び止める
瀬戸物
屋
、
とうとう
堪忍袋
の
緒
が
切れ
た
風
を
装い
、
兄貴
分
は
「
算盤
使っ
て
確かめ
て
みろ
!」
と
一喝
。
言わ
れ
た
とおり
に
勘定
し
て
みる
と
、
確か
に
計算
は
あっ
て
いる
の
だ
が
手元
を
見る
と
やはり
三
円
足り
ない
。
とうとう
パニック
に
なっ
た
瀬戸物
屋
が
「
一
荷
入り
も
持っ
て
行っ
て
ください
」
。
兄貴
分
が
「
二つ
も
いら
ない
ん
だ
」
と
言う
と
、
瀬戸物
屋
が
大きな
声
で
一言
。
「
お金
も
返す
から
!」
困っ
た
瀬戸物
屋
が
「
もうし
、
これ
な
ん
という
壺
で
ん
ねん
。
」
と
言う
と
、
兄貴
分
が
「
それ
が
こっち
の
思う
壺
や
。
」
『
時
そば
』
に
並ぶ
詐欺
師
が
主人公
の
噺
。
似
た
パターン
の
話
は
世界
各地
に
あり
、
詐欺
師
の
考え
は
東西
共通
で
ある
こと
が
うかがえる
。
また
、
詐欺
に
先立っ
て
、
駆け引き
による
合法
的
な
値切り
の
話術
が
披露
さ
れる
二
段
構え
に
なっ
て
いる
ところ
も
この
噺
の
特徴
で
ある
。
元々
は
『
壺
算用
』
という
上方
落語
で
あり
、
3
代目
三遊
亭
金
馬
が
東京
へ
移入
し
た
。
一時期
、
カタコト
の
外人
による
つり銭

が
「
時
そば
詐欺
」
と
言わ
れ
た
こと
が
ある
が
、
同じ
もの
を
二
度
数え
させ
て
相手
を
煙
に
巻い
て
しまう
やり方
は
むしろ
この
『
壺
算
』
に
近い
。
この
噺
は
「
料金
紛失
トリック
」
が
絡ん
で
おり
、
途中
の
「
二
荷
入り
の
瓶
の
勘定
は
合っ
て
いる
の
か
、
否
か
」
を
理解
する
ため
に
少し
頭
を
使う
必要
が
ある
噺
で
ある
｡
ちなみに
、
正解
は
「
勘定
は
合っ
て
い
ない
」
で
ある
｡
この
場合
、
瓶屋
の
主人
の

は
「
二
荷
入り
の
瓶
」=「
一
荷
入り
の
瓶
」+「
支払
済
の
3
円
」
と
し
て
いる
こと
に
ある
。
割引
後
の
「
二
荷
入り
の
瓶
」=「
一
荷
入り
の
瓶
」+「
3
円
」
で
いい
の
だ
が
、
この
「
一
荷
入り
の
瓶
」=「
支払
済
の
3
円
」
な
ので
、
「
二
荷
入り
の
瓶
」=「
支払
済
の
3
円
」+「
3
円
」
と
なり
、
もう
3
円
兄貴
分
から
受け取ら
なけれ
ば
なら
ない
。
しかし
、
実際
に
手元
に
ある
金
は
やはり
兄貴
分
から
受け取っ
た
「
3
円
」
で
ある
ため
、
瓶屋
の
主人
は
これ
を
大きく
勘違い
し
、
この
噺
の
オチ
に
つながっ
て
しまう
結果
と
なる
の
で
ある
｡
