『
鬼あざみ
』
は
上方
落語
の
演目
の
一つ
。
四
代目
桂
文
團
治
、
弟子
の
四
代目
桂
文
紅

三
代目
桂

。
安兵衛
は
女房
に
先立た

、
一

だ

と
暮らし

いる

家主
の
好意
で
後
添え
の
女房
お
まさ
を
貰う
。
親子
三
人
貧しく
とも
平和
な
日々
を
送っ
て
い
た
。
そんな

の

種
は
、
清吉
が

なつか
ない

で
あっ

。

に

小

」

、

、

、

き
た

に

お
母
ん
に
やら
れ
た
ん

」

。
事情
を

ない

は
子供

の

的
に
おまさ
を
なじり
、

の
中
は
大
もめ
に
なる
が
、

、

が

の

、

。

。

、
どこ
か

し
た

に
奉公
に

た
方
が
ええ
で
」
と
諭さ
れる
。
驚い
た
安兵衛
は
おまさ
に
謝罪

、
寝
て

で

て

ず

。
こうして

に
出す
。
それから

後

、

成人
し
た
清吉

安兵衛

て

。
「

も
御

で

」

て
い
た
。
「
おい
。

、
何
し
てる
ねん
。
清吉
が
帰っ
て
き
よっ
た
ん
や
。
」
「
まあ
、
清吉
」

おまさ
も
大

、
「

やろ
。
風呂
は

。

、

。

。

大金
が

に

を
抱く
。
よもや
と

た

が
、
帰っ
て
き
た
清吉
に
「
清吉
、
おのれ
は
なあ
」
と
涙
ながら
に

と
、
清吉
は
悪びれる
様子
も
なく
、
「
こう

たら
仕方
が
ねえ
。
実は

一
年
も
もた
ず

関東

れる

た
。

の

、

た

を
言い

た

。
講釈
ネタ

。

『

』

が
見
られる
。
後半

。
なお
、

の

、
お

ます
。

。

四

。

、

。
