火事
息子
（
かじ
むすこ
）
は
、
江戸
落語
の
演目
の
一つ
。
江戸
の
町
。
神田
に
ある
質屋
の
大
店
「
伊勢屋
」
の
若旦那
は
、
子供
の
頃
から
、
どういう
わけ
か
火事
が
大好き
だっ
た
。
しかし
、
それ
が
高じ
て
、
実家
を
勘当
さ
れ
た
挙句
、
臥煙
(
定
火消し
)
に
なっ
て
しまう
。
ある
風
の
強い
冬
の
日
、
「
伊勢屋
」
の
近所
で
火事
が
あっ
た
。
番頭
や
店
の
もの
は
、
蔵
の
目塗り
など
不慣れ
な
対応
で
大わらわ
。
そこ
へ
体
中
に
入れ墨
を
し
た
火消し
の
若者
が
颯爽
と
現れ
て
、
番頭
を
手助け
し
て
くれ
た
。
その
若者
が
若旦那
だっ
た
。
幸い
、
火
は
大事
に
なら
ず
に
消える
。
番頭
の
はからい
で
若旦那
と
両親
は
数
年
ぶり
に
、
気まずく
も
嬉しい
対面
を
する
こと
に
なる
。
勘当
は
さ
れ
て
も
子
は
親
を
、
親
は
子
を
思い
続ける
複雑
な
心情
を
描い
た
人情噺
。
江戸
落語
の
中
でも
大
ネタ
の
一つ
と
さ
れる
。
若旦那
の
名
は
「
藤
三郎
」
（
初代
三遊
亭
圓
右
、
6
代目
三遊
亭
圓
生
）
、
「
徳三郎
」
（
5
代目
古今
亭
志
ん
生
、
8
代目
林家
正蔵
）
、
「
徳之助
」
（
3
代目
桂
三木助
）
と
演者
によって
異なる
。
8
代目
林家
正蔵
、
6
代目
三遊
亭
圓
生
、
3
代目
桂
三木助
ら
が
得意
と
し
た
。
3
代目
桂
三木助
は
夢
で
若旦那
が
母親
に
会い
涙
を
流し
て
目
を
覚ます
という
場面
を
冒頭
に
入れ
て
い
た
が
、
これ
は
4
代目
古今
亭
志
ん
生
（
鶴本
の
志
ん
生
）
が
演じ
て
い
た
型
だ
と
いう
。
この
型
は
7
代目
（
自称
：
5
代目
）
立川談志
や
3
代目
三木助
の
弟子
で
ある
9
代目
入船亭扇橋
、
談
志
や
扇橋
の
一部
の
門弟
ら
が
踏襲
し
て
いる
。
