崇徳院
（
す
と
くい
ん
）
は
、
古典
落語
の
演目
の
一つ
。
東京
落語
における
皿
屋
（
さら
や
）
、
花見
扇
（
はなみ
おう
ぎ
）
について
も
この
項目
で
記述
する
。
元々
は
上方
落語
の
演目
で
、
後
に
東京
で
も
口演
さ
れる
よう
に
なっ
た
。
桂
文治
␣
(
初代
)
の
作
と
伝わる
。
『
崇徳院
』
は
、
小倉
百
人
一
首
77
番
の
崇徳院
の
和歌
「
瀬
を
は
やみ
␣
岩
に
せか
るる
␣
滝川
の
␣
われ
て
も
末
に
␣
あ
は
むとぞ
思ふ
」
（
詞花
和歌集
・
恋
上
。
意
＝
傾斜
の
ため
に
速く
なり
、
岩
に
当たっ
て
二

に
分かれ
て
いる
川
の
流れ
が
、
やがて
ひとつ
に
合流
する
よう
に
、
今
別れ別れ
に
なっ
て
いる
あなた
と
も
、
また
いつか
逢い

と
思っ
て
い
ます
）
を
題材
に
し

、
町人
の
恋

を
描い
た
滑稽
噺
。
初代
桂
文治
の
作
と
いわ
れ
、
後世
に
改作
を
繰り返し
現在
の
形
に
なっ
た
と
さ

。
主

演者
に
、
上方
で
は
3
代目
桂
米朝
、
3
代目
笑

亭

2

桂
春

、
2
代目
桂枝雀

桂

、
笑

松

東京

3
代目
桂
三木助

10

金原

馬

が
知ら
れる
。
30

ある

寄席

、
プログラム

時間

でき

独演

、
中入り

。
『

』

』
は
、
『

』

同様
、
和歌
を
題材
に
し

ストーリー
は
同一
で
ある
が
、
登場
人物
や
道具
など
の
ディテール
が
異なる
。
大まか

設定
は
上方
の
『
崇徳院
』
に
準じる
。
商家
の
若旦那
（
上方
で
は
作次郎
）
が
食欲
と
体力
を
失い
、
重病
に
なっ
た
。
親
旦那

、

「

で
は
治ら
ない

で
、
思い

が
叶え
ば

、

が

さ

ない

。
以下

五
郎
」
で
統一
）

に

「
座敷
へ
行っ
て
、
若旦那
に
『
思い
ごと
』
を
聞き出し
て
来い
」

。
若旦那
は
消え入り
そう
な

、

以下

。
20

「
高津

」
。

「
上野

、

に

娘

、

、

（

）

を

に
せか
るる
␣

」

、
歌
の

に

、
去っ
て

。
若旦那

、

「
われ

も
末
に
␣
あ
は
むとぞ
思ふ

、

今日
の

お

のち

お目にかかれ

、

の

会う

ず

熊
五
郎

、

3

その

捜し出せ

熊

。
熊
五
郎

人
