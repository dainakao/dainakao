。
『

改作
し

。
典型

音源

2

三遊

歌
が
これ
に
似
た
小
噺
を
し
た
もの
が
残っ
て
いる
。
明治

、
初代
柳家
小
せん
が
不潔
な
部分
を
取り除い

『
鼠
の
懸賞
』
に
直す
。
その後
、
3
代目
三遊
亭
金
馬
が
さらに
手
を
加え
て
『
藪
入り
』

する
。
天保
15

、

の
小伝馬
町
に
あっ
た

屋
「
島屋
」

、
番頭
が
小僧
（

の
意
）

性的
虐待
し
た

に

さ
せ
て
しまっ
た
。
この

に
なっ

、
この
落語
が

さ
れ

。
ただし
、

は
当時
珍しく
なかっ
た
。
ある

の
番頭
が
、
同じ
奉公人
仲間
の
小僧
を
もてあそん
で
い
た
が
、
その

として
金
を
与え
て
い
た
。
さて

住み込み

奉公人

に

。
例

小僧
も
、
両親
が
待ちわびる
実家
へ
帰っ
て
いっ
た
。
門口

立派

する
小僧
。
両親
は
わが子
の
成長
ぶり
に
感心
し
、
とりあえず
汗
を
流し
て
やろ

、
子供
を
湯屋
へ
やる
。
ところが
、
残さ
れ
た
子供
の
紙入れ
（
財布
）
の
中
を
見
た
母
は
、
中
に
大金
が
入っ
て
いる
こと

驚く
。
奉公
先
で
の
小遣い
と
は
考え
られ
ず
、
魔
が
さし
た
の

、

もみ

。
わが子

信じろ

妻
を
叱り付ける
父親

て
き
た
子供
を
殴りつけ
て
しまう
。
「
盗ん
だ
ん
じゃ
ねえ
」

息子
は
、
番頭
と
の
関係
を
説明
する
。
オチ
は
父親
「
お
釜
（
上
）
さま
の
おかげ
だ
」
